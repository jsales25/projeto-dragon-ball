# Landing Page Dragon Ball

Uma landing page dedicada ao universo de Dragon Ball, criada com HTML, CSS e JavaScript.

## Descrição

Este projeto é uma landing page simples e responsiva inspirada no anime Dragon Ball. Inclui elementos visuais, animações e interatividade básica para apresentar informações sobre o universo de Dragon Ball imerso no mundo da tecnologia.

## Estrutura do Projeto

```
landing-page-dragon-ball/
├── index.html          # Arquivo principal HTML
├── src/
│   ├── css/
│   │   ├── estilos.css     # Estilos principais
│   │   ├── reset.css       # Reset de estilos
│   │   └── responsivo.css  # Estilos responsivos
│   ├── js/
│   │   └── index.js        # Scripts JavaScript
│   ├── fonts/              # Fontes personalizadas
│   └── images/             # Imagens do projeto
```

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização e responsividade
- **JavaScript**: Interatividade e funcionalidades dinâmicas

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em um navegador web.
3. Navegue pela página para explorar o conteúdo.

## Funcionalidades

- Design responsivo para dispositivos móveis e desktop
- Animações CSS
- Interatividade com JavaScript
- Imagens e fontes otimizadas
